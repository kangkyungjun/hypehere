# HypeHere Google Play Store Marketing Assets Plan

## 📱 프로젝트 개요

**앱 이름**: HypeHere
**카테고리**: 교육 > 언어 학습
**타겟 유저**: 언어 교환을 원하는 글로벌 사용자 (Z세대, 밀레니얼)
**핵심 가치**: 진정한 언어 교환 + 실시간 소셜 네트워킹

## 🎨 1. 앱 아이콘 디자인 (512x512)

### 디자인 컨셉
**"연결과 대화의 순간"**
- 2025 트렌드 반영: 미니멀리즘 + 생동감 있는 그라데이션 + 3D 심볼리즘
- 말풍선(채팅)과 지구(글로벌)를 결합한 독창적 심볼
- 젊고 친근한 느낌의 현대적 디자인

### 컬러 팔레트
```
Primary Gradient:
- #FF6B9D (Vibrant Pink) → #C44569 (Deep Rose)
- 따뜻하고 친근한 느낌, 소셜 네트워킹 감성

Secondary:
- #4ECDC4 (Turquoise) - 언어 학습의 신선함
- #FFE66D (Sunny Yellow) - 활기와 긍정적 에너지
- #FFFFFF (White) - 깨끗한 배경

Adaptive Icon:
- Foreground: 그라데이션 말풍선 아이콘 (투명 배경)
- Background: #FF6B9D → #C44569 그라데이션
```

### 아이콘 구성 요소
1. **메인 심볼**: 둥근 말풍선 안에 지구 실루엣
2. **악센트**: 말풍선 꼬리 부분에 작은 하트/별 디테일
3. **타이포그래피**: 아이콘 자체에는 텍스트 없음 (심볼만)
4. **스타일**: 3D 느낌의 그림자/하이라이트 효과

### 기술 사양
- **크기**: 512x512 픽셀 (32비트 PNG, 알파 채널)
- **최대 파일 크기**: 1024KB
- **Safe Zone**: 중앙 66% 영역에 주요 요소 배치
- **Adaptive Icon**:
  - 포그라운드 레이어: 108x108dp (투명 배경)
  - 백그라운드 레이어: 108x108dp (단색 또는 그라데이션)

**"글로벌 연결의 생동감"**

### 구성 요소
1. **배경**:
   - 그라데이션 (#FF6B9D → #4ECDC4)
   - 추상적인 지구본/연결선 패턴 (반투명)

2. **메인 비주얼**:
   - 좌측: 앱 아이콘 (180x180, 드롭 섀도우)
   - 중앙: "HypeHere" 로고타입 (Pretendard Bold, 72pt)
   - 우측: 휴대폰 모형 속 채팅 화면 미리보기

3. **카피라이팅**:
   ```
   전 세계 친구들과 실시간 언어 교환
   ```
   - 위치: 로고 하단
   - 폰트: Pretendard Medium, 32pt
   - 컬러: 흰색

4. **악센트 요소**:
   - 말풍선, 하트, 별 아이콘 (작게 배치)
   - 국기 이모지 🇰🇷🇺🇸🇯🇵🇪🇸 (모서리에)

### 기술 사양
- **크기**: 1024x500 픽셀
- **포맷**: 24비트 PNG 또는 JPEG
- **최대 파일 크기**: 1MB
