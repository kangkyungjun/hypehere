## 통합 시나리오 테스트

### 시나리오 1: 추가전략 1 사용
1. [ ] 로또 페이지 접속
2. [ ] "추가전략 1" 버튼 클릭
3. [ ] 번호 6개 생성 확인
4. [ ] "확인" 버튼 클릭
5. [ ] 결과 모달 확인
6. [ ] "저장" 버튼 클릭
7. [ ] 저장된 번호에 "추가전략 1" 태그 확인

### 시나리오 2: 정보 모달 확인
1. [ ] 로또 페이지 접속
2. [ ] 추가전략 1 ⓘ 아이콘 클릭
3. [ ] 모달에 "추가전략 1: 빈출 번호 우선" 제목 확인
4. [ ] "상위 25개 번호 선택" 설명 확인
5. [ ] "46배 향상" 확률 정보 확인
6. [ ] X 버튼으로 모달 닫기

### 시나리오 3: 여러 전략 비교
1. [ ] 추가전략 1로 번호 생성 → 저장
2. [ ] 추가전략 2로 번호 생성 → 저장
3. [ ] 추가전략 3으로 번호 생성 → 저장
4. [ ] 추가전략 4로 번호 생성 → 저장
5. [ ] 저장된 번호 목록에서 각 전략 태그 확인

### 시나리오 4: 에러 처리
1. [ ] 네트워크 오류 시 에러 메시지 표시
2. [ ] 생성 실패 시 사용자 알림
3. [ ] 버튼 disabled 상태 정상 복구
